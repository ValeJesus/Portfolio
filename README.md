# 🌐 Portfólio Pessoal

## 📌 Sobre o Projeto

Este projeto é uma **página de portfólio pessoal** desenvolvida para apresentar minhas informações profissionais de forma simples e organizada.

O portfólio reúne:

* Meus **certificados**
* Meu **currículo**
* Link para meu **GitHub**
* Minhas **redes sociais**

O objetivo é centralizar tudo em um só lugar e facilitar o acesso ao meu perfil profissional.

---

## 🛠️ Tecnologias Utilizadas

O portfólio será desenvolvido utilizando:

* **HTML** → Estrutura da página
* **CSS** → Estilização e layout
* **JavaScript** → Interatividade e funcionalidades

---

## 📄 Estrutura da Página

A página será composta por uma única tela (one page), contendo as seguintes seções:

### 🧑‍💼 Sobre Mim

* Breve apresentação pessoal
* Objetivo profissional

---

### 📜 Certificados

* Lista de certificados obtidos em cursos
* Nome do curso e instituição
* Possibilidade de links para os certificados

---

### 📑 Currículo

* Informações profissionais resumidas
* Botão para **download do currículo em PDF**

---

### 💻 GitHub

* Link direto para meu perfil no GitHub
* Destaque para projetos e repositórios

---

### 🌍 Redes Sociais

* Links para redes sociais profissionais
* Exemplos: LinkedIn, Instagram, X (Twitter), entre outros

---

## 🎯 Objetivo do Projeto

* Criar uma identidade profissional online
* Facilitar o compartilhamento do currículo
* Mostrar conhecimentos em desenvolvimento web
* Servir como portfólio para oportunidades acadêmicas e profissionais
